# Meeting Transcript Summary

Summarize a publicly accessible Google Docs meeting transcript into a clear,
structured recap inside an OpenHands chat.

## Triggers

This skill is activated by keywords such as:

- `meeting transcript`
- `google docs transcript`
- `meeting notes summary`
- `meeting highlights`
- `action items from transcript`

## Features

- Fetches a publicly accessible Google Doc without requiring login
- Prefers plain text export for cleaner transcript parsing
- Extracts highlights, decisions, action items, blockers, and open questions
- Supports short, medium, and detailed recap formats
- Handles long transcripts by chunking and synthesizing intermediate summaries

## Prerequisites

A Google Docs link that is publicly accessible from the sandbox, for example
`Anyone with the link can view` or a published-to-web document.

## Quick Start

Ask OpenHands:

> "Summarize this meeting transcript from Google Docs and pull out the
> highlights, decisions, and action items: <public-google-doc-url>"
